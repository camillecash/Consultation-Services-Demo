# TeckyConsult Website Demo

A responsive, two-page concept site for Kenneth Gyan's IT consulting business.

## Pages

- `index.html` — home, about, services, process, podcast teaser, and consultation form
- `podcast.html` — podcast positioning, future episode concepts, and listening-platform links

## Brand direction

The visual system uses muted navy, sea-glass teal, sage, warm ivory, and a restrained coral accent. The custom mark represents a guided technical path: structure at the top, a clear route through the middle, and a destination point.

## Easy launch updates

1. Replace `hello@teckyconsult.com` with Kenneth's preferred email.
2. Replace the four general podcast-platform URLs with the show's profile URLs.
3. Connect `#consult-form` in `script.js` to a scheduler, CRM, or form service.
4. Update the sample podcast topics when real episodes are available.
5. Add privacy and terms pages before collecting live visitor information.

## Structure and expansion

The demo has no build step and uses semantic HTML, shared CSS, and lightweight JavaScript. Sections can be split into additional pages without changing the visual system. A future CMS or framework migration can reuse the existing content, colors, layout, and assets.
